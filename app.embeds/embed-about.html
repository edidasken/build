The Wellspring — Built for Where the Internet Ends

The vast majority of the global Church is not in a coffee shop with fast Wi-Fi. It is in a village. In a home. In a place where the internet arrives once a week, if at all. FlockOS was built to serve them too — not as an afterthought, but as a first-class citizen of the kingdom.

The Wellspring is FlockOS's local-first data engine. A pastor can export their entire church database — all members, prayer requests, groups, events, pastoral notes, and history — into a single encrypted file. That file lives on their device. FlockOS reads from it. No internet required.

📥
Export once

When connected, export your full church data as an encrypted .xlsx file. All 200+ data tabs. One file.

→
📵
Work offline

Load the file into FlockOS. The full app — members, care, groups, events — runs entirely in the browser with no connection.

→
🔄
Sync when ready

Next time you're online — whether that's tomorrow or next month — export again. Your changes come with you.

256-Bit AES-GCM Encryption. PBKDF2 Key Derivation. Your data is yours.

The Wellspring vault uses industry-standard AES-GCM encryption with PBKDF2 (100,000 iterations) key derivation from a PIN that is never stored. Church member data — names, addresses, prayer requests, pastoral notes — is encrypted at rest in the browser’s IndexedDB. Only the right PIN unlocks it. No cloud vendor can read it. No breach can expose it. The pastor’s flock is protected even on a shared device in a remote community.
